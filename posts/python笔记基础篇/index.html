Python笔记（基础篇）

February 8, 2022

  • 编程基础
  • python
  • 笔记

一. 基础语法

1.编码

源码文件以 utf-8 编码，所有字符串都是 unicode 字符串。还可以用码源文件指定不同的编码：

# _*_ coding: utf-8 _*_
2.标识符
  • 第一个字符必须是字母或下划线
  • 标识符其他部分由字母，数字和下划线组成
  • 标识符对大小写敏感
3.注释

Python 中单行注释以 #开头，多行注释可以用多个#号，还有’’’,""";

4.行与缩进

python 最具特色的就是属于使用缩进行来表示代码块，不需要使用大括号**{}**。同时缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的空格缩进数。

if Ture
    print ('Ture')
else
    print ('False')
5.多行语句

python 通常是一行写完一条语句，但是如果语句太长，可以使用 \ 来实现多行语句。

在[],{},()中的多行语句，不需要使用反斜杠 \ ,例如：

total = ['item_one','item_two','item_three',
        'item_four','item_five']
6.数字（Number）类型

python 中数字有四种类型：整数，布尔型，浮点数和复数

  • int（整数），如 1，只有一种整数类型 int，表示为长整型，没有 python2 中的 long
  • bool（布尔），True，False
  • float（浮点数），如 1.23，3E-2
  • complex（复数），如 1+2j ，1.1+2.2j
7.字符串（String）

字符串可以用 **+运算符连接在一起，用***运算符重复

反斜杠可以用来转义，使用r可以让反斜杠不发生转义

python 中的字符串不能改变

单引号和双引号使用完全相同

8.空行

函数之间或类方法之间用空行分隔，表示一段新的代码的开始。类和函数入口之间也用一行空行分开，以突出函数入口的开始。

9.等待用户输入

实例：

input ("\n\n 按下enter键后退出。")
10.同一行显示多条语句

python 可以在同一行中使用多条语句，语句之间使用 ; 分割。

import sys ; x = 'chen' ; sys.stdout.write (x + '\n')
11.多个语句构成代码组

缩进相同的一组语句构成一个代码块，我们称之为代码组。

像 if,while,de 和 class 这样的复合语句，首先以关键字开始，以冒号**(:)**结束，该行之后的一行或多行代码构成代码组。

首行及后面的代码组称为一个子句（clause）

if expession
   suite
elif expression
   suite
else :
   suite
12.print 输出

print 默认输出是换行的，如果实现不换行需要在变量末尾加上 end= “”

x = 'a'
y = 'b'
#换行输出
print (x)
print (y)

print('-------------')
#不换行输出
print (x, end = " ")
print (y, end = " ")
print ()
13.import 和 from…import

在 python 中，用import 或者 from…import来导入相应模块。

  • 将整个模块（somemoudule）导入，格式为 import somemodule
  • 从某个模块中导入某个函数，格式为：from somemodule import somefunction
  • 从某个模块中导入多个函数，格式为： from somemodule import fristfunc, seconedfunc,thirdfunc
  • 将某个模块中的全部函数导入，格式为： from somemodule import
14.命令行参数

程序可以执行一些操作来查看一些基本信息，Python 可以使用-h 参数查看各参数帮助信息

二.基本数据类型

python 中的变量不需声明，每个变量在使用前都必须赋值，只有变量赋值之后，这个变量才会被创建。

而且变量就是变量，它没有类型，我们所说的 “ 类型 ” 就是变量所指的内存中对象的类型。

等号 = 用来给变量赋值，左边是变量名，右边是存储在变量中的值。

counter = 100
miles   = 1000.0
name    = "陈星"

print (counter)
print (miles)
print (name)
1.多个变量赋值

同时为多个变量赋值。

a = b = c = 1

创建一个整数类型，值为 1，从后往前赋值，三个变量被赋予相同的数值。

也可以为多个对象指定多个变量

a, b , c = 1, 2, 'chen'
2.标准数据类型

python 中一共有六个数据类型：

  • Number（数字）
  • String（字符串）
  • List（列表）
  • Tuple（元组）
  • Set（集合）
  • Dictionary（字典）

其中，前三个是不可变数据，后三个是可变数据

Number（数字）

python 支持 int，float，bool，complex（复数）

内置的 type 函数可以查询变量所指的对象类型。此外还可以用 isinstance 来判断

二者之间的区别：

  • type（）不会认为子类是一种父类类型。
  • isinstance（）会认为子类是一种父类类型。

注意 ：bool 是 int 的子类，True 和 False 可以和数字相加，True= =1， False==0，但是可以通过is来判断类型。

字符串（String）

字符串用单引号或者双引号括起来，同时使用反斜杠\转义特殊字符。

字符串的截取语法格式如下：

变量[头下标：尾下标]，例如：

str = 'chenxing121'
print(str)  # 输出字符串
print(str[0:-1])  # 输出第一个到倒数第二个的所有字符
print(str[4])  # 输出第五个字符
print(str * 2)  # 输出两边字符串
print(str + 'error')  # 字符串的拼接

​ 反斜杠\转义特殊字符，如果不想让反斜杠发生转义，可以在任意字符串前面加一个原始字符串

需要注意的是：

  • 1、反斜杠可以用来转义，使用 r 可以让反斜杠不发生转义。
  • 2、字符串可以用+运算符连接在一起，用*运算符重复。
  • 3、Python 中的字符串有两种索引方式，从左往右以 0 开始，从右往左以-1 开始。
  • 4、Python 中的字符串不能改变。
List（列表）

列表可以完成大多数集合类的数据结构实现，元素的种类可以不相同，甚至可以包含列表(嵌套)

列表是写在方括号 [ ] 之间，用逗号分隔开。

**注意：**和字符串一样，列表同样可以被索引和截取，列表被截取后返回一个包含所需元素的新列表。 格式：变量[头下标：尾下标]

实例：

list = ['chen', 123344, 22.3, 'xing', '星星']
tinylist = [123, 'xingxingshuai']

print(list)
print(list[3])  # 输出列表第四个元素
print(list[1:3])  # 输出第二个元素到第三个元素
print(list[2:])  # 输出输出从第三个元素开始的所有元素
print(tinylist * 2)
print(list + tinylist)  # 连接列表

注意：

  • 1、List 写在方括号之间，元素用逗号隔开。

  • 2、和字符串一样，list 可以被索引和切片。

  • 3、List 可以使用+操作符进行拼接。

  • 4、List 中的元素是可以改变的。
Tuple（元组）

与列表类似，不同之处在于元组的元素不能修改，元组写在小括号**（ ） **里，元素之间用逗号隔开。

实例：

tuple = ('abcd', 342, 3.3333, 'chen')
tinytuple = (1234, 'chen')

print(tuple)
print(tuple[0])
print(tuple[1:3])
print(tuple[2:])
print(tinytuple * 3)
print(tuple + tinytuple)

注意：

  • 1、与字符串一样，元组的元素不能修改。
  • 2、元组也可以被索引和切片，方法一样。
  • 3、注意构造包含 0 或 1 个元素的元组的特殊语法规则。
  • 4、元组也可以使用+操作符进行拼接。
Set（集合）

集合是由一个或数个形态各异的大小整体组成的，构成集合的事务或者对象称作元素或者是成员。

基本功能是进行成员关系测试和删除重复元素。

可以使用大括号或者是**set ()**函数创建集合，**注意：创建一个空集必须使用set ()而不是大括号，因为{ }**是用来创建一个空集合。

创建格式:

parame = {value01 , value02 , ...}
#或者
set (value)

实例：

sites = {'goole', 'chen', 'taobao', 'zhihu'}

# 输出集合，重复的元素被自动取掉
print(sites)

# 成员测试
if 'xing' in sites:
    print('chen 在集合中')
else:
    print('chen 不在集合中')

# set可以进行集合运算
a = set('chenxing')
b = set('shidashuaige')

print(a)

print(a - b)  # a 和 b 的差集
print(a | b)  # a 和 b 的并集
print(a & b)  # a 和 b 的交集
print(a ^ b)  # a 和 b 中不存在的元素
Dictionary（字典）

列表是有序的对象集合，字典是无序的对象集合。

两者之间的区别在于：字典当中的元素是通过键来存取的，而不是通过偏移存取。

字典是一种映射类型，字典用 { } 标识，它是一个无序的 键(key) : 值(value) 的集合。

键(key)必须使用不可变类型。

在同一个字典中，键(key)必须是唯一的。

dict = {}
dict['one'] = '1 - chen'
dict[2] = '2 - xing'

tinydict = {'name': 'chen', 'code': 1, 'site': 'chenxing.com'}


print(dict['one'])    # 输出键为 'one' 的值
print(dict[2])        # 输出键为 2 的值
print(tinydict)  # 输出完整的字典
print(tinydict.keys())  # 输出所有键
print(tinydict.values())  # 输出所有值

注意

  • 1、字典是一种映射类型，它的元素是键值对。
  • 2、字典的关键字必须为不可变类型，且不能重复。
  • 3、创建空字典使用 { }

三.数据转换

数据类型的转换，一般情况下你只需要将数据类型作为函数名即可。

数据类型转换可以分为两种:

  • 隐式类型转换（自动）
  • 显式类型转换（需要使用类型函数来转换）

隐式类型转换

两种不同类型的数据进行运算，较低数据类型（整数）就会转换为较高数据类型（浮点数）以避免数据丢失。

实例：

num_int = 123
num_flo = 1.22

num_new = num_int + num_flo

print('datatype of num_int:', type(num_int))
print('datatype of num_flo:', type(num_flo))

print('value of num_new:', num_new)
print('datatype of num_new', type(num_new))

输出结果：

num_int 数据类型为: <class 'int'>
num_flo 数据类型为: <class 'float'>
num_new: 值为: 124.23
num_new 数据类型为: <class 'float'>

显式类型转换

在显式类型转换中，用户将对象的数据类型转换为所需的数据类型。 我们使用 int()、float()、str() 等预定义函数来执行显式类型转换。

int() 强制转换为整型，float() 强制转换为浮点型，str() 强制转换为字符串类型。

整型和字符串类型进行运算，就可以用强制类型转换完成：

num_int = 123
num_str = '456'

print('num_int 数据类型为：', type(num_int))
print('类型转换前，num_str数据类型为：', type(num_str))

num_str = int(num_str)  # 强制转换为整型
print('类型转换后，num_str数据类型为：', type(num_str))

num_sum = num_int + num_str

print('num_int 与 num_str 相加结果为：', num_sum)
print('sum数据类型为：', type(num_sum))

结果：

num_int 数据类型为： <class 'int'>
类型转换前，num_str数据类型为： <class 'str'>
类型转换后，num_str数据类型为： <class 'int'>
num_int 与 num_str 相加结果为： 579
sum数据类型为： <class 'int'>
